# ✏️ 인덱스 메모 (Index Memo App) v3.0 

> **(1.0,2.0은 MVP로 따로 README.md 파일이 제공되지 않습니다!)**
> 
> **모니터 가장자리에 자석처럼 쇽! 숨었다 나타나는 나만의 스마트 인덱스 포스트잇**
> 
> 깔끔한 파스텔톤 디자인과 강력한 멀티 모니터 지원, 그리고 컴퓨터를 껐다 켜도 완벽하게 기억하는 스마트 자동 저장 기능을 탑재한 가볍고 강력한 윈도우 전용 메모 프로그램입니다.

---

## ✨ 핵심 기능 (Key Features)

### 1. 📍 자석식 인덱스 슬라이딩 탭
* 화면 가장자리(상/하/좌/우)에 자석처럼 밀착되어 깔끔하게 숨겨집니다.
* 마우스 왼쪽 버튼으로 인덱스 탭(`✏️`)을 클릭하면 부드러운 애니메이션과 함께 메모지가 스르륵 나타나고 닫힙니다.

### 2. 🖥️ 완벽한 멀티 모니터 지원
* 듀얼, 트리플 등 여러 개의 모니터를 사용하는 환경에서도 완벽히 작동합니다.
* 마스터 설정(`⚙️`) 메뉴를 통해 원하는 모니터로 한 번에 메모지 세트를 통째로 이동시킬 수 있습니다.

### 3. 💾 스마트 실시간 자동 저장 & 복구
* 메모 내용을 수정하거나, 창 크기를 조절하거나, 색상을 바꿀 때마다 배경에서 **실시간으로 자동 저장**됩니다.
* 프로그램을 종료하거나 컴퓨터가 갑자기 꺼져도 작성 중이던 모든 데이터가 안전하게 보존되며, 다음 실행 시 원 상태 그대로 완벽 복구됩니다.

### 4. 💻 윈도우 시작 시 자동 실행
* 컴퓨터를 켤 때마다 프로그램을 일일이 켜지 않아도 됩니다.
* 마스터 설정(`⚙️`) 메뉴에서 클릭 한 번으로 윈도우 시작 프로그램에 안전하게 등록 및 해제할 수 있습니다.

### 5. 🔤 전역 글꼴(폰트) 커스텀 및 가독성 최적화
* 내 PC에 설치된 모든 폰트, 크기, 스타일(굵게/기울임)을 실시간으로 일괄 적용할 수 있습니다.
* 글꼴 크기를 크게 키워도 내용이 잘리지 않도록 텍스트 영역 내에 **상하 스크롤바가 자동으로 활성화**됩니다.

### 🎨 개별 색상 및 자유로운 위치 편집
* 5가지 감성적인 파스텔톤 색상 스위치 제공으로 메모의 중요도나 종류에 맞게 꾸밀 수 있습니다.
* 인덱스 탭을 **마우스 우클릭 드래그**하여 위아래 또는 좌우로 자유롭게 순서를 바꾸고 배치할 수 있습니다.

## ⚙️ 시스템 요구사항 (System Requirements)

* **운영체제:** Windows 10 / Windows 11 (32비트 및 64비트 전원 지원)
* **런타임 환경:** .NET Framework 4.0 이상 (윈도우 10/11 환경에는 기본 내장되어 있어 별도 설치가 필요 없습니다.)
* **용량:** 약 100KB 내외의 초경량 단독 실행 파일

---

## 🚀 다운로드 및 실행 방법 (How to Run)

1. 배포처에서 `IndexMemo_v1.0.exe` 파일을 다운로드합니다.
2. 원하는 폴더(예: 바탕화면, 문서 폴더 등)에 넣어줍니다.
3. 실행 파일을 **더블 클릭** 하면 즉시 작동합니다.
   * *※ 최초 실행 시 같은 폴더에 환경 설정 및 데이터 저장을 위한 `memo_data.txt` 파일이 자동으로 생성됩니다.*

## ⚠️ 최초 실행 및 사용시 주의사항 (Windows SmartScreen 경고)

본 프로그램은 개인이 개발하여 디지털 서명 인증서를 등록하지 않은 순수 유틸리티입니다. 이로 인해 최초 실행 시 윈도우에서 아래와 같이 **파란색 경고창(SmartScreen)** 이 뜰 수 있습니다. **안심하고 실행하셔도 됩니다.**
또한 현재 버전에서는 데이터 안정성을 위해 일부 특수문자(`|`) 입력을 제한하고 있습니다!

### 💡 경고창 통과 방법:
1. 파란색 안내창이 뜨면 당황하지 마시고 화면 상단의 **[추가 정보]** 글자를 클릭합니다.
2. 우측 하단에 새로 나타나는 **[실행]** 버튼을 누르면 정상적으로 프로그램이 켜집니다.
3. 이후 두 번째 실행부터는 경고창 없이 즉시 실행됩니다.

## 🛠️ 조작 방법 꿀팁 (Tips & Tricks)

* **메모지 펼치기/접기:** 인덱스 탭(`✏️`)을 마우스 **왼쪽 버튼**으로 클릭하세요.
* **메모지 순서 바꾸기:** 인덱스 탭(`✏️`)을 마우스 **우클릭한 상태로 드래그**하면 가장자리를 따라 위치가 부드럽게 이동합니다.
* **메모지 크기 조절:** 메모지가 펼쳐진 상태에서 **우측 하단 가장자리 모서리**를 마우스로 잡고 늘리거나 줄일 수 있습니다.
* **프로그램 완전 종료:** 본 프로그램은 투명 캔버스 방식이라 작업 표시줄에 뜨지 않습니다. 완전히 종료하고 싶으실 때는 마스터 설정(`⚙️`) 버튼을 누른 뒤 맨 아래의 **[❌ 프로그램 완전히 종료]** 메뉴를 이용해 주세요.
